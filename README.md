# İlanPazar

Modern ve kullanıcı dostu bir ilan paylaşım platformu.  
Bu proje, kullanıcıların ilan oluşturabildiği, görüntüleyebildiği ve yönetebildiği bir web uygulaması olarak geliştirilmiştir.

## Özellikler

- Kullanıcı kayıt ve giriş sistemi
- Güvenli oturum yönetimi
- İlan oluşturma ve listeleme
- Responsive ve modern arayüz
- Tailwind CSS ile tasarım
- MySQL veritabanı desteği
- Next.js tabanlı hızlı yapı

## Kullanılan Teknolojiler

- Next.js
- React
- Tailwind CSS
- Radix UI
- MySQL
- bcryptjs
- iron-session

## Kurulum

Projeyi bilgisayarınıza klonlayın:

```bash
git clone https://github.com/iiremozturkk/ilanpazar.git
```

Proje klasörüne girin:

```bash
cd ilanpazar
```

Gerekli paketleri yükleyin:

```bash
npm install
```

Geliştirme sunucusunu başlatın:

```bash
npm run dev
```

Tarayıcıda açın:

```text
http://localhost:3000
```

## Proje Yapısı

```text
components/   -> Arayüz bileşenleri
pages/        -> Sayfalar
public/       -> Statik dosyalar
styles/       -> CSS dosyaları
hooks/        -> Custom React hook'ları
```

## Amaç

Bu proje, modern web teknolojileri kullanılarak geliştirilen bir ilan pazarı uygulaması örneğidir.  
Frontend ve backend yapısını birlikte kullanarak tam işlevsel bir platform oluşturmayı hedeflemektedir.
